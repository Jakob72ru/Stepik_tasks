Здесь выбраны некоторые интересные задачи с пройденных мною курсов на площадке Stepik.org

- Задачи находятся в папке tasks. В названии файлов с задачами в скобках указана тема к которой относится задача.
- Каждая задача имеет описание, ссылку на описание на площадке Stepik.org и мое авторское решение.
- Все решения выработаны и написаны мной без помощи LLM чат-ботов. 
- Решения прошли тестирование на площадке и получили подтверждение корректности решения.
